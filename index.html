hexo发布新文章的步骤

发表于 2018-08-03

1、新建markdown文档

2、创建文章名称

鼠标右键打开Git bash here

输入 hexo new “文章名称”（要有空格，””要用英文）

2、创建文章的具体内容

打开本地的logs文件夹

找到source文件夹打开

打开_posts，打开刚生成的markdown文件，粘贴，编辑，保存。

3、本地测试

hexo s

打开浏览器检查效果

Ctrl + C 终止

4、发布

hexo clean && hexo g && hexo d

打开浏览器检查发布效果，如有问题，进入第二步里修改，然后执行第三四步。
